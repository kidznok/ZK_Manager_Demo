# ZK Manager — demo online

Statyczna wersja demonstracyjna przeznaczona do publikacji przez GitHub Pages.

## Publikacja

1. Utwórz na GitHubie nowe publiczne repozytorium, np. `zk-manager-demo`.
2. Wgraj do głównego katalogu repozytorium całą zawartość tego folderu — plik `index.html` musi być na najwyższym poziomie.
3. Otwórz w repozytorium **Settings → Pages**.
4. W sekcji **Build and deployment** wybierz **Deploy from a branch**.
5. Wybierz gałąź **main**, katalog **/(root)** i kliknij **Save**.
6. Po kilku minutach demo będzie dostępne pod adresem `https://NAZWA-UZYTKOWNIKA.github.io/zk-manager-demo/`.

Wersja online korzysta tylko z przykładowych danych zapisanych lokalnie w przeglądarce. Nie ma dostępu do plików komputera ani bazy wersji Windows.
